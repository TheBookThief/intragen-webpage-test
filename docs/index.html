InTraGen

Trajectory-controlled Video Generation for Object Interactions

About the Research

InTraGen focuses on controlling video generation using trajectories to model realistic object interactions. The proposed method emphasizes diverse and dynamic interactions, offering insights for applications in animation and simulation.

Read the full paper on arXiv.

Key Features

  • Trajectory-based control
  • Realistic object interactions
  • Generative models for video synthesis
